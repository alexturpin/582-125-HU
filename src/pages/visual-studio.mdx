## Trucs et astuces Visual Studio Code

### Lien de téléchargement

Gratuit à partir du site web de <a href="https://code.visualstudio.com/">Visual Studio Code</a>.

### Sauvegarde automatique des paramètres

1. _File_
1. _Preferences_
1. _Turn on Settings Sync..._
1. Vous identifier avec votre compte Microsoft

### Formattage automatique des fichiers lors de sauvegarde

1. _File_
1. _Preferences_
1. _Settings_
1. Rechercher pour `editor.formatOnSave`
1. Cocher la case _Editor: **Format On Save**_

### Génération de paragraphes de Lorem Ipsum

Raccourci [Emmet](https://css-tricks.com/need-lorem-ipsum-inside-html/) afin de générer 5 paragraphes

```
p*5>lorem
```

### Validation de code HTML directement à partir de votre éditeur

Installer l'extension [_W3C Web Validator_](https://marketplace.visualstudio.com/items?itemName=CelianRiboulet.webvalidator).

1. Cliquer sur l'icône d'extension dans la barre de gauche (les 4 blocs)
2. Chercher pour `celianriboulet.webvalidator`
3. Cliquer sur _Install_
